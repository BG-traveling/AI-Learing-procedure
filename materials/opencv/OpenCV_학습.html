학습 순서

OpenCV를 공부할 때 머릿속에 잡아야 하는 큰 흐름

OpenCV는 단순히 함수 이름을 외우는 것보다, “이미지를 숫자 배열로 보고 → 원하는 부분을 강조/변환/검출/추적한다”는 흐름으로 이해하면 훨씬 쉽습니다.

1단계: 영상 정규화
이미지는 결국 픽셀 값 배열입니다. 밝기, 대비, 색공간, 마스크, 히스토그램을 조절해서 컴퓨터가 보기 좋은 형태로 바꿉니다.
2단계: 기하학적 변환
이미지의 좌표를 이동, 회전, 확대/축소, 아핀 변환, 투시 변환으로 바꿉니다. 즉, 픽셀의 위치를 새 좌표로 매핑합니다.
3단계: 특징 검출
엣지, 코너, 윤곽선, 직선/원 같은 눈에 띄는 특징을 찾아냅니다. 객체 인식 전처리의 핵심입니다.
4단계: 추적
템플릿 매칭, 특징자 매칭, MeanShift, CamShift, Optical Flow로 영상 속 객체나 움직임을 따라갑니다.
01 영상 정규화

핵심 개념

영상 정규화 단원은 디지털 영상의 표현, RGB/HSV 색공간, 픽셀 연산, 마스크, 히스토그램 정규화/평활화를 다룹니다.

디지털 영상

아날로그 신호를 샘플링, 양자화, 인코딩해서 숫자 배열로 만든 것이 디지털 영상입니다. 픽셀 값은 보통 0~255 범위를 가집니다.

RGB와 HSV

RGB는 화면 표시에는 직관적이지만 조명 변화에 약합니다. HSV는 색상, 채도, 명도를 분리하므로 색 기반 분할에 유리합니다.

마스크

마스크는 특정 영역만 선택하기 위한 흑백 이미지입니다. 값이 0이면 제외, 0이 아니면 적용입니다.

히스토그램

픽셀 밝기 분포를 보는 도구입니다. 특정 구간에 몰린 값을 넓게 펼치면 대비가 좋아집니다.

02 기하학적 변환

핵심 개념

기하학적 변환은 픽셀 값 자체보다 픽셀의 좌표를 바꾸는 작업입니다.

변환의미대표 함수
회전중심점 기준으로 이미지를 돌림getRotationMatrix2D, warpAffine
확대/축소좌표 비율을 늘리거나 줄임resize
Affine평행선이 유지되는 2D 변환getAffineTransform
Perspective원근감을 반영하여 평행선이 깨질 수 있음getPerspectiveTransform
03 특징 검출

핵심 개념

특징은 이미지에서 “다른 부분과 구별되는 변화 지점”입니다. 밝기가 급격히 바뀌는 곳은 엣지, 두 방향으로 변화가 큰 곳은 코너로 볼 수 있습니다.

Sobel

밝기 변화율을 근사하는 필터입니다. x방향, y방향 변화량을 구해서 경계를 찾습니다.

Canny

노이즈 제거, 그래디언트 계산, 비최대 억제, 이중 임계값 처리를 거쳐 얇고 정확한 엣지를 찾습니다.

Hough Transform

엣지 정보를 파라미터 공간으로 변환하여 직선이나 원 같은 도형을 검출합니다.

04 Optical Flow / 추적

핵심 개념

추적은 한 프레임에서 찾은 객체나 특징점이 다음 프레임에서 어디로 이동했는지 찾는 작업입니다.

방법핵심
템플릿 매칭작은 템플릿 이미지를 원본에서 슬라이딩하며 유사도 비교
MeanShift색상 히스토그램 기반으로 가장 비슷한 영역을 반복 탐색
CamShiftMeanShift 개선. 크기와 회전 변화까지 적응
Lucas-Kanade특징점이 프레임 사이에서 조금 움직인다고 가정하고 추적
Dense Optical Flow모든 픽셀의 움직임 벡터를 계산

코드 실습 템플릿

아래 코드는 수업에서 바로 실습할 수 있는 최소 형태입니다. 이미지 경로만 바꿔서 실행해보세요.

이미지 불러오기 + RGB 변환

import cv2
import matplotlib.pyplot as plt

image = cv2.imread("sample.jpg")

# OpenCV는 BGR, matplotlib은 RGB 기준
rgb = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)

plt.imshow(rgb)
plt.axis("off")
plt.show()

기억 포인트: cv2.imread 결과가 None이면 경로가 틀린 것입니다.

HSV 색공간 + 마스크

import cv2
import numpy as np

image = cv2.imread("sample.jpg")
hsv = cv2.cvtColor(image, cv2.COLOR_BGR2HSV)

# 예: 빨간색 계열은 HSV에서 두 구간으로 나누는 경우가 많음
lower1 = np.array([0, 80, 80])
upper1 = np.array([10, 255, 255])
lower2 = np.array([170, 80, 80])
upper2 = np.array([180, 255, 255])

mask1 = cv2.inRange(hsv, lower1, upper1)
mask2 = cv2.inRange(hsv, lower2, upper2)
mask = mask1 + mask2

result = cv2.bitwise_and(image, image, mask=mask)

블러 / 엣지 검출

gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

blur = cv2.GaussianBlur(gray, (5, 5), 0)
edge = cv2.Canny(blur, 100, 200)

plt.imshow(edge, cmap="gray")
plt.axis("off")
plt.show()

기억 포인트: Canny 전에 GaussianBlur를 적용하면 노이즈에 덜 민감해집니다.

회전 변환

h, w = image.shape[:2]
center = (w // 2, h // 2)

matrix = cv2.getRotationMatrix2D(center, 45, 1.0)
rotated = cv2.warpAffine(image, matrix, (w, h))

기억 포인트: 변환 행렬을 만들고, 실제 적용은 warpAffine이 합니다.

히스토그램 평활화

gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
equalized = cv2.equalizeHist(gray)

plt.hist(gray.ravel(), bins=256, range=[0, 256])
plt.show()

템플릿 매칭

image = cv2.imread("source.jpg", cv2.IMREAD_GRAYSCALE)
template = cv2.imread("template.jpg", cv2.IMREAD_GRAYSCALE)

result = cv2.matchTemplate(image, template, cv2.TM_CCOEFF_NORMED)
min_val, max_val, min_loc, max_loc = cv2.minMaxLoc(result)

print("가장 유사한 위치:", max_loc)
print("유사도:", max_val)
