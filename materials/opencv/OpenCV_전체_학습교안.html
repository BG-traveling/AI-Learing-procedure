1. 이미지 기초: OpenCV에서 이미지는 숫자 배열이다

OpenCV는 이미지를 사람이 보는 그림이 아니라 NumPy 배열로 다룹니다.

이미지의 shape

컬러 이미지는 보통 (세로, 가로, 채널) 구조입니다.


image = cv2.imread("sample.jpg")
print(image.shape)  # 예: (480, 640, 3)
print(image.ndim)   # 컬러 이미지는 3차원
print(image.dtype)  # 보통 uint8, 값 범위 0~255

BGR과 RGB

OpenCV는 이미지를 기본적으로 BGR 순서로 읽습니다. 그런데 matplotlib은 RGB 순서로 보여주기 때문에 변환하지 않으면 색이 이상하게 보일 수 있습니다. 


bgr = cv2.imread("sample.jpg")
rgb = cv2.cvtColor(bgr, cv2.COLOR_BGR2RGB)

plt.imshow(rgb)
plt.axis("off")
plt.show()
쉽게 이해하기

이미지는 작은 칸, 즉 픽셀들의 모음입니다. 흑백 이미지는 각 칸에 0~255 숫자 하나가 들어 있고, 컬러 이미지는 각 칸에 B, G, R 세 개 숫자가 들어 있습니다. 그래서 OpenCV에서 이미지를 바꾼다는 것은 결국 배열 안의 숫자를 바꾸는 것입니다.

자주 하는 실수: cv2.imread()로 읽은 이미지를 바로 plt.imshow()하면 색이 뒤집혀 보일 수 있습니다. matplotlib으로 볼 때는 거의 항상 cv2.COLOR_BGR2RGB 변환을 생각하세요.

2. 색공간·HSV·마스크

색상 추출 실습의 핵심은 HSV로 변환 → lower/upper 범위 설정 → mask 생성 → bitwise_and 적용입니다.

HSV는 왜 쓸까?

RGB/BGR은 빨강·초록·파랑 값이 조명에 따라 같이 흔들립니다. HSV는 색상(H), 채도(S), 밝기(V)를 분리해서 표현하므로 “빨간색만 찾기” 같은 작업에 더 편합니다.

값의미OpenCV 범위
HHue, 색상0~179
SSaturation, 채도0~255
VValue, 밝기0~255

mask와 result의 차이

mask는 흑백 지도입니다. 범위 안이면 255, 범위 밖이면 0입니다.

result는 그 마스크를 원본 이미지에 적용해서 원하는 색만 남긴 이미지입니다.


hsv = cv2.cvtColor(image, cv2.COLOR_BGR2HSV)

lower_red = np.array([0, 100, 100])
upper_red = np.array([20, 255, 255])

mask = cv2.inRange(hsv, lower_red, upper_red)
result = cv2.bitwise_and(image, image, mask=mask)
mask 비유: 마스크는 “구멍 뚫린 종이”입니다. 흰색 부분은 구멍이 뚫려 원본이 보이고, 검은색 부분은 가려져 검정으로 사라집니다.

빨간색은 왜 두 구간으로 잡을까?

OpenCV의 Hue는 0~179 원형 구조입니다. 빨간색은 0 근처에도 있고 179 근처에도 있기 때문에 두 마스크를 합치는 경우가 많습니다. 


lower_red1 = np.array([0, 80, 50])
upper_red1 = np.array([10, 255, 255])

lower_red2 = np.array([170, 80, 50])
upper_red2 = np.array([180, 255, 255])

mask1 = cv2.inRange(hsv, lower_red1, upper_red1)
mask2 = cv2.inRange(hsv, lower_red2, upper_red2)

red_mask = cv2.bitwise_or(mask1, mask2)

3. 히스토그램: 이미지의 밝기 분포 보기

히스토그램은 “0~255 밝기 값이 각각 몇 개나 있는지”를 세는 그래프입니다. 어두운 이미지인지, 밝은 이미지인지, 대비가 약한지 확인할 때 사용합니다.

히스토그램 계산


gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

hist = cv2.calcHist(
    [gray],      # 이미지
    [0],         # 채널 번호
    None,        # 마스크 없음
    [256],       # 0~255 총 256칸
    [0, 256]     # 값 범위
)

plt.plot(hist)
plt.show()

히스토그램 평활화

평활화는 한쪽에 몰린 픽셀 값을 넓게 퍼뜨려 대비를 높이는 방법입니다. 어둡거나 흐릿한 이미지에서 디테일이 더 잘 보이게 만들 수 있습니다. 


gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
equalized = cv2.equalizeHist(gray)

plt.imshow(equalized, cmap="gray")
plt.axis("off")
plt.show()

4. 필터링·이진화·모폴로지

필터링은 주변 픽셀을 이용해 이미지를 부드럽게 하거나 노이즈를 줄이는 작업입니다. 모폴로지는 이진 이미지에서 흰색 영역을 깎거나 부풀려 형태를 정리하는 작업입니다.

기법함수언제 쓰나?쉽게 말하면
평균 블러cv2.blur전체적으로 부드럽게주변 값을 단순 평균
가우시안 블러cv2.GaussianBlur자연스럽게 노이즈 제거중앙 픽셀에 더 큰 가중치
중앙값 블러cv2.medianBlur소금후추 노이즈 제거주변 값 중 중앙값 선택
침식cv2.erode작은 흰색 노이즈 제거흰색 영역을 깎음
팽창cv2.dilate끊어진 흰색 영역 연결흰색 영역을 부풀림
열기MORPH_OPEN작은 노이즈 제거침식 후 팽창
닫기MORPH_CLOSE작은 구멍 메우기팽창 후 침식

블러 예시


image = cv2.imread(image_path)
image = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)

blur = cv2.blur(image, (7, 7))
gaussian = cv2.GaussianBlur(image, (7, 7), 0)
median = cv2.medianBlur(image, 7)

이진화 + 모폴로지 예시


gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

_, binary = cv2.threshold(
    gray, 127, 255, cv2.THRESH_BINARY
)

kernel = np.ones((3, 3), np.uint8)

opening = cv2.morphologyEx(binary, cv2.MORPH_OPEN, kernel)
closing = cv2.morphologyEx(binary, cv2.MORPH_CLOSE, kernel)
모폴로지는 보통 흑백 이진 이미지에서 효과가 잘 보입니다. 컬러 이미지에 바로 적용하기보다 GRAY → threshold → morphology 순서로 생각하세요.

5. 크롭·리사이즈·회전·Affine·Perspective

기하학적 변환은 이미지 픽셀의 위치를 바꾸는 작업입니다. 즉, “색을 바꾸는 것”이 아니라 “픽셀이 어디로 이동할지”를 정합니다.

크롭과 리사이즈

image[세로범위, 가로범위]로 원하는 부분만 잘라낼 수 있습니다.


# crop: 세로 250~300, 가로 400~600 영역
crop = image[250:300, 400:600]

# resize: dsize는 (가로, 세로) 순서
small = cv2.resize(image, dsize=(600, 500))

회전

회전은 중심점, 각도, 스케일을 정한 뒤 warpAffine으로 적용합니다.


h, w, _ = image.shape
center = (w / 2, h / 2)

matrix = cv2.getRotationMatrix2D(
    center,  # 회전 중심
    250,     # 각도
    1        # 스케일
)

rotated = cv2.warpAffine(image, matrix, (w, h))

Affine 변환

Affine은 이동, 회전, 크기 조절, 기울이기를 포함합니다. 2x3 행렬에서 t1, t2는 x/y 이동입니다. 


affine_matrix = np.array([
    [1,  0.3, 50],
    [-0.3, 1, 50]
]).astype(np.float32)

result = cv2.warpAffine(image, affine_matrix, (w, h))

Perspective 변환

Perspective는 사다리꼴처럼 찍힌 영역을 정면 사각형처럼 보정할 때 사용합니다. 문서 스캔, 카드 인식, 표지판 보정에 자주 쓰입니다. 


src_pts = np.float32([
    [200, 100],
    [400, 100],
    [110, 300],
    [580, 300]
])

dst_pts = np.float32([
    [0, 0],
    [299, 0],
    [0, 399],
    [299, 399]
])

M = cv2.getPerspectiveTransform(src_pts, dst_pts)
result = cv2.warpPerspective(image, M, (300, 400))

6. 복습문제 기반 실습

Drive의 복습 문제 노트북에서 다루는 연습 흐름을 과제형으로 정리했습니다.

실습 1. 포인트 컬러 필터

목표: 빨간색은 컬러로 남기고, 나머지는 흑백으로 바꾸기


def color_filter(filename):
    bgr = cv2.imread(filename)
    hsv = cv2.cvtColor(bgr, cv2.COLOR_BGR2HSV)

    lower1 = np.array([0, 80, 50])
    upper1 = np.array([10, 255, 255])
    lower2 = np.array([170, 80, 50])
    upper2 = np.array([180, 255, 255])

    mask1 = cv2.inRange(hsv, lower1, upper1)
    mask2 = cv2.inRange(hsv, lower2, upper2)
    red_mask = cv2.bitwise_or(mask1, mask2)

    gray = cv2.cvtColor(bgr, cv2.COLOR_BGR2GRAY)
    gray_bgr = cv2.cvtColor(gray, cv2.COLOR_GRAY2BGR)

    result = np.where(red_mask[:, :, None] == 255, bgr, gray_bgr)
    return result

실습 2. 특정 색상만 다른 색으로 바꾸기

목표: 마스크로 색을 바꿀 픽셀 위치를 찾고, HSV의 H 값만 변경하기


def change_color_pixel(filename, ranges, new_hue):
    image = cv2.imread(filename)
    hsv = cv2.cvtColor(image, cv2.COLOR_BGR2HSV)

    mask = np.zeros(hsv.shape[:2], dtype=np.uint8)

    for lower, upper in ranges:
        temp = cv2.inRange(hsv, lower, upper)
        mask = cv2.bitwise_or(mask, temp)

    y, x = np.where(mask == 255)

    for yi, xi in zip(y, x):
        hsv[yi, xi, 0] = new_hue

    result = cv2.cvtColor(hsv, cv2.COLOR_HSV2BGR)
    return result

실습 3. 희미한 부분 선명하게 만들기

목표: 이미지 일부를 crop한 뒤 threshold/adaptiveThreshold로 선명도를 비교하기


brain = image[:, :270]
gray = cv2.cvtColor(brain, cv2.COLOR_BGR2GRAY)

# 여러 임계값을 비교
for value in [20, 40, 60, 80, 100, 120, 140, 160, 180]:

# 주변 픽셀까지 고려하는 적응형 이진화
adaptive = cv2.adaptiveThreshold(
    gray,
    255,
    cv2.ADAPTIVE_THRESH_GAUSSIAN_C,
    cv2.THRESH_BINARY,
    77,

실습 4. 노이즈가 심한 이미지 전처리

목표: threshold 후 모폴로지 연산으로 동그란 덩어리들을 더 잘 연결하기


gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

_, binary = cv2.threshold(gray, 127, 255, cv2.THRESH_BINARY)
